JAVA SCRIPT - JOÃO ASAFE

Java Script e Java são a mesma coisa?

Java é uma linguagem de programação orientada a objetos, usada principalmente
para aplicativos de grande porte e sistemas Android.
JavaScript é uma linguagem de script usada principalmente para tornar páginas
web interativas. Ou seja, não são a mesma coisa.


Quando surgiu o Java Script?

JavaScript foi criado no ano de 1995 a pedido da empresa Netscape


Quem foi o responsável pela criação do Java Script?

Brendan Eich foi o programador responsável pela criação do JavaScript.
Eich criou a linguagem em apenas 10 dias, com o objetivo de adicionar
interatividade às páginas da web, tornando-as dinâmicas.


Qual o caminho trilhado pelo reponsável pela criação do Java Script?

Brendan Eich, criador do JavaScript, teve uma trajetória marcante na indústria de tecnologia.
Ele começou sua carreira na Netscape Communications em 1995, onde desenvolveu o JavaScript para
tornar as páginas web mais interativas. Após a aquisição da Netscape pela AOL, Eich foi um dos fundadores
da Mozilla Foundation em 1998, onde contribuiu para o desenvolvimento do navegador Firefox e promoveu
os padrões abertos para a web, incluindo o aprimoramento do JavaScript.
Em 2015, Eich co-fundou a Brave Software, uma empresa dedicada a melhorar a privacidade na web.
A Brave criou um navegador que bloqueia anúncios e rastreadores, e introduziu um sistema de recompensas
baseado em criptomoeda (BAT - Basic Attention Token). Hoje, a Brave continua sendo uma empresa ativa no mercado,
oferecendo uma alternativa focada em privacidade ao navegador tradicional.
Portanto, Brendan Eich segue influente no desenvolvimento de tecnologias para a web, com Brave ainda sendo
uma empresa relevante na área de privacidade online.
